# LINE Hub

WordPress 的 LINE 整合中樞外掛

## 功能特色

- ✅ LINE 登入與 OAuth 認證
- ✅ WordPress 帳號自動註冊
- ✅ Email 擷取與驗證
- ✅ 多場景訊息通知（訂單、出貨、會員）
- ✅ Webhook 統一管理
- ✅ 第三方外掛整合（FluentCart、BuyGo Plus 1）
- 🔮 未來擴充（WooCommerce、FluentCRM、外部 SaaS）

## 系統需求

- WordPress 6.5+
- PHP 8.2+
- MySQL 5.7+ / MariaDB 10.3+

## 安裝

1. 上傳外掛到 `wp-content/plugins/line-hub/`
2. 在 WordPress 後台啟用外掛
3. 前往「LINE Hub」→「入門」開始設定
