# Plan 10-01: REST API 文件與 curl 範例

## Objective

擴充開發者 Tab 的 REST API 端點清單，讓每個端點都有清晰的說明、參數表格、可直接複製執行的 curl 範例。

## Requirement Coverage

- **DEV-01**: 開發者 Tab 顯示完整 REST API 端點清單（含 curl 範例）

## Current State

開發者 Tab（`tab-developer.php`）已有：
- API Key 管理區塊（產生/撤銷）
- REST API 端點清單（4 個端點，含 curl 範例）
- WordPress Hooks 區塊（3 Actions + 2 Filters）
- Shortcodes 區塊
- PHP 範例區塊

## Changes

### Step 1: 擴充 DeveloperTab class（class-developer-tab.php）

**修改** `includes/admin/tabs/class-developer-tab.php`:
- 新增 `get_api_endpoints()` 方法，回傳結構化的端點資料
- 新增 `get_hooks_data()` 方法，回傳結構化的 Hook 資料
- 傳遞結構化資料給 view 模板

### Step 2: 重新設計 tab-developer.php view

**修改** `includes/admin/views/tab-developer.php`:
- 補齊遺漏的端點：`GET /users/lookup`

- curl 範例加入「複製」按鈕（利用現有 `line-hub-copy-btn` class）
- 回應範例區塊
- API Key 取得方式的提示說明
